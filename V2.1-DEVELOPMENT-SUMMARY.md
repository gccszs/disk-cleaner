## 项目概述

**disk-cleaner** 是一个跨平台磁盘空间管理工具集，具备智能分析、安全清理、实时监控等核心功能。

### 核心特性
- 🔍 智能磁盘分析 - 快速定位占用空间的大文件和目录
- 🧹 安全垃圾清理 - 自动清理临时文件、缓存、日志等
- 📊 实时监控告警 - 持续监控磁盘使用情况
- 🌍 跨平台支持 - 完美支持 Windows、Linux、macOS
- ⚡ 零依赖 - 纯 Python 实现，无需安装第三方库

### 项目结构
```
disk-cleaner/
├── skills/disk-cleaner/       # 技能包目录
│   ├── SKILL.md               # 完整功能指南
│   ├── scripts/               # 可执行脚本（8个）

│   └── docs/                  # 文档（10个文件）

├── tests/                     # 测试套件（244个测试）

```

### ⚡ 新功能：渐进式扫描

**问题**: 大磁盘（500GB+）完整扫描需要数小时，用户失去耐心

**解决方案**: 创建专用渐进式扫描脚本

#### analyze_progressive.py（新增）
```bash

python scripts/analyze_progressive.py --sample

# 渐进式扫描（30秒）

# 文件数量限制
python scripts/analyze_progressive.py --max-files 50000
```

**核心功能**:
- 🔍 快速采样模式 - 1秒估算扫描时间
- 📊 渐进式扫描 - 30秒获取部分结果
- 🔄 实时反馈 - 每2秒更新进度
- ⏸️ 可中断 - Ctrl+C 获取部分结果

#### analyze_disk.py（增强）

### Lint 规则

**配置**:
```python
# black
line-length = 100

# isort
profile = "black"

# flake8
max-line-length = 100
extend-ignore = E203,E231,W503
# 对于 skills/ 脚本额外忽略
extend-ignore = F401,F541,F841,F811
```

**忽略的警告说明**:
- **F401**: 未使用的导入（部分是为了兼容性）
- **F541**: f-string缺少占位符（普通字符串即可）
- **F841**: 未使用的局部变量
- **F811**: 重复导入（argparse在main中重新导入）
- **E203**: 冒号前空格（Black兼容）
- **E231**: 冒号后空格（Black兼容）
- **W503**: 二元运算符换行（Black兼容）

## 文件归档功能讨论

**状态**: 📋 需求收集完成，待设计实现

**讨论文档**: `~/Desktop/pj_md/disk-cleaner-归档功能设计讨论.md`

### 核心需求

**用户场景**:
1. 桌面整理 - "帮我整理桌面"
2. 文件归档 - 将同类文件收集到新建目录
3. 智能分类 - 按类型、时间、语义自动分组

### 已确定的设计决策

| 决策点 | 选择 | 说明 |
|--------|------|------|
| **主要目的** | B（文件整理分类）+ D（综合方案） | 按类型、日期、项目维度整理 |
| **触发方式** | C（智能建议模式） | Agent建议→用户确认→执行 |
| **整理粒度** | B（混合级别） | 文件+文件夹都可整理 |
| **分类规则** | A+B+C | 扩展名+时间+语义分析 |
| **安全机制** | C（Dry-run+确认） | 必须用户明确确认 |
| **确认界面** | A（结构化摘要） | 清晰、简洁、易理解 |

### 安全设计原则

1. **用户明确同意** - 任何移动操作必须确认
2. **Dry-run优先** - 默认显示计划
3. **结构化展示** - 清晰易理解的计划
4. **可撤销** - 保留索引支持撤销
5. **保护敏感文件** - 自动排除
6. **低阅读成本** - 5-10秒理解
